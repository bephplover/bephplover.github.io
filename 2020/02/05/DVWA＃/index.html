这道题是关于csrf的，也是最基本的一种输入新密码，并输入确认密码，然后点击按钮就会发现url：

4.File inclusion


<?php

// The page we wish to display
$file = $_GET[ 'page' ];

?>

原理很简单传入page的会直接被包含进来。

那么怎么利用呢，我们分别将include.php换成file1.php,file2.php,file3.php

就可以分别看到三个php中的内容了。

这里其实需要php将配置点上allow_url_include才可默认包含。

文件包含还有一个点就是被包含的文件会默认当做php文件来执行，也就是如果有上传漏洞和文件包含漏洞打组合拳就可以getshell了。
