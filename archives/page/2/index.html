Git

简介

Git是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。git是Linus Torvalds为了帮助管理Linux内核开发而开发的一个开放源码的版本控制软件。
Git与常用的版本控制工具CVS，Subversion等不同，它采用了分布式版本库的方式，不必服务器端软件支持。
Git不仅仅是个版本控制系统，也是一个内容管理系统（CMS），工作管理系统。
